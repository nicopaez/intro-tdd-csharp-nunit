Fizz Bazz Kata
==============

Reglas:
* f(n no es multimplo de 3 ni de 5) = n
* f(n es multiplo de 3) = fizz
* f(n es multiplo de 5) = buzz
* f(n es multimplo de 3 y de 5) = fizzbuzz

Escenarios:
>* f(1) = 1
>* f(2) = 2
> * f(3) = fizz
> * f(6) = fizz
> * f(5) = buzz
> * f(10) = buzz
> * f(15) = fizzbuzz
> * f(30) = fizzbuzz
